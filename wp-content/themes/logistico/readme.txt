Logistico is a smart & dynamic transport and logistics WordPress theme that can build a startup, corporate businesses, online companies website with minimal of efforts. 

== Description ==
Logistico WordPress Theme is made for logistic services, transportation companies, packers and movers, corporate businesses, online companies in mind. It can fit as a small company website as well as a medium size business site. Logistico is fantastically designed to suit any blog/news or as a e-commerce website with WooCommerce. It is equiped with Elementor page builder and Themepaw Companion to make it more flexible and robust. Cross browser supported and Full responsive design to fit into any display sized devices and mobiles.  WPML ready and 100% SEO friendly code can boost your site with Logistico WordPress Theme. 


== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

Or

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Search Logistico in search bar right side of the screen.
3. After found Logistico install and then activate and enjoy.


== Documentation ==
= Plugins and Installation =

Logistico Essential Plugins and Installation. For using all features and to get the demo like pages, you have to install and activate all recommended plugins.
- Elementor Page Builder
- Themepaw Companion
- Contact Form 7

Install above essential plugins in 2 easy steps.

Step 1: After Installing the theme, You will see a notification at the top of the page asking to install the recommended plugins. Install these recommended plugins from there. It will automatically activate the plugins for you. If it doesn't, follow the Step 2.

Note: This might take couple of minutes.
Step 2:Then click on Activate link after installation under each plugin title and enjoy these plugins.


= One Click Demo Import =
We already intrigate with "One click demo import" plugin for you. So, install and active all plugins from Appearance -> "Install Plugins" after sucessfully activated all plugin you will se an new option called "Import Demo Data" under Appearance. Click on "Import Demo Data" menu then you will find "Import Demo Data" Button bottom of the page click and import one click feature. 


= Manually Demo Content Import =
Before start import make sure you have install and active all the recommended plugin we have been listed for you.

Import Sample Data: To get the pre built pages, you will need to import the xml demo data. From Dashboard, Go to Tools>Import>Run Importer at WordPress section. If you don't have "WordPress Importer" plugin installed, it will ask you to install the plugin. Then download xml file from https://files.themepaw.com/logistico/content.xml then navigate the "content.xml" file from your download folder or where it downloaded. Click "Upload file and Import" button.

Widget Import: To import theme demo widgets, You need this plugin installed "Widget Importer & Exporter". Once Installed, you will find a new Menu under Tools named "Widget Importer & Exporter". Then download wie file from https://files.themepaw.com/logistico/widgets.wie then navigate the "widgets.wie" file from your download folder or where it downloaded and click Import Widgets button.

Customizer Data Import: To import theme customizer option data, You need this plugin installed "Customizer Export/Import". Once Installed, you will find a new Menu under Appearance>Customize>Export/Import. Then download dat file from https://files.themepaw.com/logistico/customizer.dat then navigate the "customizer.dat" file from your download folder or where it downloaded and click Import button.
